# has-many-count
Laravel Nova HasMany Count Field

## Requirements

This Nova field requires Nova 2.

## Installation

```bash
composer require sashalenz/has-many-count
```

## Usage

```php
use Sashalenz\HasManyCount\HasManyCount;

HasManyCount::make(__('Posts'), 'posts'),
```
